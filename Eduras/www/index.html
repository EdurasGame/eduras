About

Screenshot of deathmatch mode

Eduras? is a online multiplayer strategy shooter.

Playing with your friends, you can compete for items, strength and kills.

Features

  • Many maps varied in size, opportunities and difficulty
  • No player limit
  • Three game modes: Deathmatch, Team-Deathmatch, Capture-The-Flag
  • Play with your friends via internet
  • Platform independent (Java)
  • Spectator mode

Tutorial

Starting and playing Eduras? should be self-explaining. If you need any help, this page should help you. If not, feel free to contact us.

Setup

Starting server with custom port

After starting the server application, the server automatically listens on port 4387. To change the port, you can provide a custom port as first parameter.
